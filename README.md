# C Doubly Linked List

This is an implementation of a doubly linked list in the C programming language.

The current features of the implementation are:

- create dll
- registerCompareData
- registerFreeData
- registerPrintData
- head
- tail
- curr
- size
- hasNext
- next
- hasPrev
- prev
- traverse dll
- searchHeadToTail
- searchTailToHead
- searchHeadAndTail
- search dll
- pushHead
- pushTail
- addBefore
- addAfter
- freeNode
- delete
- popHead
- popTail
- clear dll
- reverse dll
- sort dll
- print dll

The file sample.c shows some usage examples.
The file interactive.c provides a little command line like interface to test the features.
